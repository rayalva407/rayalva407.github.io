My Projects

Home Swap

A website made on Ruby on Rails for trading homes. Implements user authentication and enables users to create an account, post their homes, and select other user's homes to make trips on the dates of their choosing.

Cycles

A web application made with HTML/CSS/JS that enables women to track their menstrual cycle by entering menstrual start date. This tool will generate cycle and fertility predictions based on previous cycles. User can add and delete cycle entries.
