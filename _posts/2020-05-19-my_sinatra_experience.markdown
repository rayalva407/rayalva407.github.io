Working on this project was very eye opening. It was a great learning experience and I didn't know I would be able to create a project like this before. I am so excited to learn more!

At first coming up with an idea for the project proved to be really difficult. I didn't know what I wanted to do at all. I think it is because I was just thinking too hard about it. I decided to keep things very simple so that I would worry more about learning the process. I didn't want to make things more difficult than they had to be. My project idea was inspired by one of my friends who loves poetry. I decided to make a poetry vault where a user could make an account and keep all the poems they wanted in it.

When I began my project I first thought of what my models would be. I knew that there would be two models. A poem model and a user model. This was very straightforward and now all I had to think about was what my views were going to be like. I started by imagining what a user would be looking at first when they were already signed into their account. That is why I first started with the index view page. After that it was really easy to make progress. I knew that a user would have to be able to create, view, edit, and delete their poems. Creating views with that purpose in mind was very easy. After I was done with that part of my project I worked on a way to create users. I added a gem called 'bcrypt'.  This allows my page to keep user's passwords secure. This makes it very easy to authenticate the user so that they can only see their own poems. 

Looking back at this project I think I found it to be quite difficult. The concepts were easy to understand but once I tried putting them into practice there were a lot of obstacles I had to go over before finishing my project. Overall though, I thought it was a very satisfying experience and it only made me more eager to learn. My love for programming has grown more thanks to this project.
