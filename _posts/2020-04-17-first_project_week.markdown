The end of project week is officially over. I have completed my first project. It feels great to have accomplished this. I had never created a CLI application in my life so this feels like a great accomplishment. At first I felt like I would not be able to do it but here I am now. When I first started to think about my project I had an idea to make an application that would search for music events in your area. As I was working on it I didn't feel like I was really enjoying the process. The idea didn't seem fun enough for me so I ended up leaving the project all together. After a few hours of brainstorming and googling I found the Breaking Bad API. Being a fan of the show I immediately knew that I wanted to do something with that. I ended up building an app called Breaking Quotes. This application shows you quotes from the popular show. It was such a fun idea and I had fun making it. I was able to use everything I learned about object oriented programming to create this app. It felt like it was easier to apply my knowledge because I was really having fun! I know that my program is not perfect, but I know that I will continue to work on it and make it better. I actually can't wait for the next project week.
